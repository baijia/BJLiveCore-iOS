– playingUserDidUpdate:old:

用户开关音、视频

- (BJLObservable)playingUserDidUpdate:(nullable BJLUser *)now old:(nullable BJLUser *)old

Parameters

now

新用户信息
old

旧用户信息

Discussion

  • 某个用户主动开关自己的音视频时发送此通知、不包含意外掉线等情况
  • 正在播放的视频用户 关闭视频时 videoPlayingUser 将被设置为 nil、同时发送此通知
  • 进教室后批量更新 playingUsers 时『不』发送此通知
  • 音视频开关状态通过 BJLUser 的 audioOn、videoOn 获得
